BLE RSSI Various Static Configurations

This dataset contains ~30 second collects in various static configurations. Data was collected in various environments, with different phone carrying positions, and with different user rotations relative to the measurement device. Additionally, multiple measurement devices were used. All collects were done with 6' separation between the transmitting device and receiving device.

How to Submit a Dataset

If you have a dataset you are interested in seeing on this website please choose an option below to proceed.

Option 1 (preferred): Using Github Repositories

Please use this option if you are familiar with setting up a Github repository and if your dataset does not contain any single files that are larger than 4GB.

  1. Create a new repository using the PACT Template repository by following this link: https://github.com/mitll/PACTTemplateDataset/generate
  2. Add your dataset to this new repository. You may (and should) alter the README.md file. Also feel free to alter the DISTRIBUTION.md or LICENSE.md files but you should not remove them.
  3. Make an issue on the main PACT repository here: https://github.com/mitll/PACT/issues
    • tag your issue with the label Task: Dataset Submission
    • in your issue's description, include a link to the repository you created in Step 1
  4. The PACT Team will review your issue and attached dataset and will add it to the website or provide feedback as necessary

Option 2: Without Github Repositories

If you do not wish to use Github or your dataset is too large for Github's large file share (LFS) feature, please email the PACT Datasets and Evaluation team at pactdatasets@gmail.com with the dataset name and a brief description of how it was collected. Please either attach your dataset directly to the email or else provide a link to where it can be accessed by the PACT Team.
